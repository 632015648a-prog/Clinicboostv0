# Out of Scope — por ahora

## Fuera de alcance inmediato
Hasta cerrar el piloto asistido NO se prioriza:
- reescribir la arquitectura,
- abrir nuevos módulos grandes,
- vender como completos los Flows 02-07,
- dashboards avanzados de BI,
- automatizaciones sofisticadas no esenciales,
- integraciones adicionales no críticas,
- observabilidad enterprise completa,
- nuevas capas de producto que no bloquean demo/piloto.

## Ejemplos de cosas que no deben entrar “porque sí”
- motor complejo de reglas v2
- sistema avanzado de campañas
- panel completo de configuración fina
- analítica de cohortes avanzada
- multicanal más allá del caso actual
- rediseño visual completo del frontend

## Regla de decisión
Si una tarea:
1. no bloquea demo,
2. no bloquea staging,
3. no bloquea piloto,
4. y no corrige un bug serio,
entonces normalmente va al backlog, no al sprint actual.
